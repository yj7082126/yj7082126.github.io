해당 포스트는 Y.Mirsky, W.Lee 의 [The Creation and Detection of Deepfakes: A Survey]([https://www.semanticscholar.org/paper/The-Creation-and-Detection-of-Deepfakes%3A-A-Survey-Mirsky-Lee/97ac037b7589d9d0f2cb5109aa2b095ad06067fd](https://www.semanticscholar.org/paper/The-Creation-and-Detection-of-Deepfakes%3A-A-Survey-Mirsky-Lee/97ac037b7589d9d0f2cb5109aa2b095ad06067fd))를 바탕으로 합니다.

오늘은 과거부터 현재까지의 딥페이크 모델들을 정리하여 여러 모델들의 종류와 근간 기술, 트렌드, 발전 방향등에 대해 기술하고자 합니다.

현재 생산적 딥러닝 모델들의 결과물은 실제 결과물과 구분하기 어려울 정도로 발전했습니다. 2018년도를 기점으로 해당 기술들을 비윤리적인 목적으로 사용하는것이 굉장히 쉽다는 것이 밝혀진 이후, 소위 *딥페이크*라고 불리는 이미지/오디오/비디오 생산 딥러닝 모델들은 눈에 띄는 속도로 발전해왔습니다. 최근에는 딥페이크로 일반인/유명인사들의 명예를 훼손하거나 가짜 뉴스를 만드는 일로 부정적인 방향으로 이슈화가 되기도 했습니다.

## 딥페이크란?

> **심층신경망 기술 (Deep Neural Network)  에 의해 만들어진, 실제처럼 보이는 시청각 매체**

일반적으로 딥페이크는, 인공지능으로 제작되어 사람을 소

용어 정리 
* $`s`$ : source identity (Expression / Mouth / Identity 등을 **전달하는** 주체)
	* $`x_{s}`$ : source identity image
* $`s`$ : target identity (Expression / Mouth / Identity 등을 **전달받는** 주체)
	* $`x_{t}`$ : target identity image
*  $`x_{g}`$ : generated identity image.

## Technical Background

* **Encoder-Decoder Architecture**
* **Convolutional Neural Network**
* **Recurrent Neural Network**
* **Generative Adversarial Network**
* **Image-to-Image Translation**
	* **[pix2pix](https://www.semanticscholar.org/paper/Image-to-Image-Translation-with-Conditional-Isola-Zhu/8acbe90d5b852dadea7810345451a99608ee54c7)**
		* Enables translations from one image domain to another
		* Generator : generates $`x_g`$ from input visual context $`x_c`$
			* U-Net : Encoder-Decoder CNN w/ skip connections, which enables to produce high fidelity imagery by bypassing compression layers.
		* Discriminator : discriminates between $`(x, x_c)`$ and $`(x_g, x_c)`$
	* Improvement : **[pix2pixHD](https://www.semanticscholar.org/paper/High-Resolution-Image-Synthesis-and-Semantic-with-Wang-Liu/f0a0c0f0d6a7ff53abea40a8c0c678ed570bf851)**
		* Generate High Resolution imagery with better fidelity
	* Improvement : **[CycleGAN](https://www.semanticscholar.org/paper/Unpaired-Image-to-Image-Translation-Using-Networks-Zhu-Park/c43d954cf8133e6254499f3d68e45218067e4941)**
		* Image Translation thru unpaired training
		* Cycle of 2 GANs to convert images from one domain to another
